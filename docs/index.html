devc

CLI tool for generating ready-to-use development containers for any project

Quick install:

curl -fsSL https://samitox4.github.io/devc/install.sh | bash
🚀
Instant Setup
Generate a complete dev environment in seconds
📦
Multiple Templates
Node.js, Android, React Native, Java, Rust, Go, Python & more
🔄
Auto Updates
Templates stay up to date automatically

Node.js + React Native + Android

Java 17 + Maven
laravel
PHP 8.3 + Composer
rust
Rust + Cargo
go
Go 1.22
python
Python 3.12 + pip

Usage

devc gen nodejs --name my-project
Generate a Node.js devcontainer
devc gen react-native --name my-app
Generate a React Native devcontainer
devc list
List all available templates
devc update
Update templates from repository
